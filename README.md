### Tic-Tac-Toe
*   **Files:** `tic-tac-toe/tic-tac-toe.c`
*   **Description:** A grid-based puzzle game for two players competing to align symbols.
*   **Usage:** Classic tic-tac-toe game where each player tries to get their symbol three times in a row to win. Using the position numbers provided, i.e, 1 to 9, both players can place their allotted symbol (“x” or “o”) on any empty spot. When the grid is filled, but no player can secure three of their own symbols in a row, the game is drawn.
